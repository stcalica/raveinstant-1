## Requirements

- Node 4.4.7

### Installing Dependencies

Install all npm dependencies.
```sh
npm install
```

Initiate webpack.
```sh
npm run build:dev
```

Initiate nodemon.
```sh
npm start
```

OPTIONAL (Recommended): Run Webpack and Nodemon at once:
```sh
npm run build:start
```
